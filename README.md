
 # Portfolio Website

A clean and responsive personal portfolio website showcasing my skills, projects, and contact information ,for learning purpose.

## About

This is my personal portfolio website built as part of my learning journey in web development. I'm Zahid Khan, currently in my 4th year at UET Peshawar, and this project demonstrates my skills in HTML, CSS, and web design.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI** - Clean and professional interface with gradient colors
- **Smooth Navigation** - Easy-to-use navigation menu
- **Skills Section** - Showcases technical skills with interactive cards
- **Projects Gallery** - Display of completed and ongoing projects
- **Contact Information** - Easy ways to get in touch

## Technologies Used

- HTML5
- CSS3
- Responsive Design
- Git & GitHub

## Sections

1. **Header** - Introduction with name and title
2. **Navigation** - Quick links to all sections
3. **About Me** - Brief introduction and background
4. **Skills** - Technical skills and expertise
5. **Projects** - Portfolio of completed work
6. **Contact** - Ways to connect with me

## Installation & Setup

To run this project locally:

1. Clone the repository
```bash
git clone https://github.com/zahidkhan1081/zahidkhan1081portfolio.git
```

2. Navigate to the project directory
```bash
cd zahidkhan1081portfolio
```

3. Open `index.html` in your browser
```bash
# On Windows
start index.html

# On Mac
open index.html

# On Linux
xdg-open index.html
```
